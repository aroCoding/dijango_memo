메모 CRUD 설명

생성 (Create)
새로운 메모 추가

제목과 내용을 입력하여 새로운 메모를 생성할 수 있습니다.

읽기 (Read)
메모 목록 보기

저장된 메모를 목록 형태로 볼 수 있으며, 각 메모의 제목, 생성 날짜, 내용이 표시됩니다.

업데이트 (Update)
기존 메모 수정

수정 버튼을 클릭하여 기존 메모의 내용을 업데이트할 수 있습니다.

삭제 (Delete)
메모 삭제

삭제 버튼을 클릭하여 더 이상 필요하지 않은 메모를 제거할 수 있습니다.
